## Project Overview
室内巡检机器人，四轮差速驱动，搭载激光雷达和 IMU — a diff drive robot powered by stm32f407.
## Hardware Configuration

- **left_front_wheel**: dji-m3508 via drv8301, CAN ID 0x201

- **right_front_wheel**: dji-m3508 via drv8301, CAN ID 0x202

- **left_rear_wheel**: dji-m3508 via drv8301, CAN ID 0x203

- **right_rear_wheel**: dji-m3508 via drv8301, CAN ID 0x204


- **lidar**: rplidar-a3 on UART (/dev/ttyUSB0)

- **imu**: bmi088 on SPI (SPI2)
