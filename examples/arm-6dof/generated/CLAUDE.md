## Project Overview
桌面级六轴协作机械臂，前 3 关节力矩电机直驱，腕部 3 关节智能舵机 — a arm 6dof robot powered by stm32f407.
## Hardware Configuration

- **joint_1_base**: tmotor-ak80-9 via drv8301, CAN ID 0x101

- **joint_2_shoulder**: tmotor-ak80-9 via drv8301, CAN ID 0x102

- **joint_3_elbow**: tmotor-ak80-9 via drv8301, CAN ID 0x103

- **joint_4_wrist_roll**: dynamixel-xm430 via drv8301, CAN ID 0x104

- **joint_5_wrist_pitch**: dynamixel-xm430 via drv8301, CAN ID 0x105

- **joint_6_wrist_yaw**: dynamixel-xm430 via drv8301, CAN ID 0x106


- **base_imu**: bmi088 on SPI (SPI2)

## Build Commands
```bash
# Firmware
pio run                    # Compile
pio run --target upload    # Flash to MCU
# ROS 2
colcon build --packages-select arm_manipulator_bringup
source install/setup.bash
ros2 launch arm_manipulator_bringup bringup.launch.py
# Simulation
docker compose up          # Start Gazebo + ROS 2
```
